Docker 常用命令

更新时间：2022-09-20 20:14:50标签：

制作镜像

docker build -f ./deploy/Dockerfile --progress plain -t image/name .

启动容器

docker run -p 9008:80 -d image/name

进入镜像

docker exec -it 容器id bash

查看日志

docker logs 容器id -f --tail 100

释放无用资源

docker system prune -f
