Docker 运行 mysql

更新时间：2023-02-22 15:57:53标签：dockermysql

下载镜像

docker pull mysql:latest

启动镜像

$ docker run -d --name mysql -p 3307:3306 -e MYSQL_ROOT_PASSWORD=123456 mysql

命令行

连接数据库

mysql -P 3307 -h 127.0.0.1 -u root -p

查看所有数据库

show databases;

使用数据库

use xxx;

显示所有表

show tables;

查看表结构

show columns from xxx;
