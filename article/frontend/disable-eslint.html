如何临时禁用eslint

更新时间：2022-10-11 18:24:48标签：

禁用本文件的所有eslint规则

1/* eslint-disable */
2console.log('JavaScript debug log');
3console.log('eslint is disabled now');

禁用后启用

1/* eslint-disable */
2console.log('JavaScript debug log');
3console.log('eslint is disabled now');
4/* eslint-enable */
5console.log('eslint is enable now');

禁用指定规则

1/* eslint-disable no-console, no-control-regex */
2console.log('JavaScript debug log');
3console.log('eslint is disabled now');

禁用本行

console.log('eslint is disabled for the current line'); // eslint-disable-line

禁用下一行

1// eslint-disable-next-line no-console
2console.log('eslint is disabled for the current line');
