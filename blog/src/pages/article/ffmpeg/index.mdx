## 将m3u8转为mp4
```bash
ffmpeg -i xxx.m3u8 -f rawvideo -bsf h264_mp4toannexb -vcodec copy out.mp4
```

## 将电脑录屏推送到服务器（屏幕直播）
```bash
ffmpeg.exe -f dshow -framerate 30 -i video=screen-capture-recorder -vcodec libx264 -tune zerolatency -preset ultrafast -fflags nobuffer -flags low_delay -probesize 32 -analyzeduration 0 -threads 6 -f mpegts udp://127.0.0.1:1234
```
