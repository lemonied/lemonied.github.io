# 制作好安装windows 11系统的U盘

1. 打开微软官网 https://www.microsoft.com/zh-cn/software-download/windows11  ，点击`创建Windows 11安装媒体`下面的`立即下载`

2. 下载后双击运行

3. 选择中文（简体）、windows版本选择家庭版或专业版

4. 选择`U盘`，点击`下一页`（此时电脑可以插入U盘准备了）

5.  当前会显示您已经插入的U盘，并会删除U盘里的数据，所以如果有重要数据，请提前备份好 ，然后点`下一页`

6. 此时会开始下载 windows 11 镜像，请耐心等待，直至完成。

7. 制作成功之后 ，点击 `完成` 即可

# 备份好硬盘中的重要数据

> 注：部分机器需要开启`TPM`功能，需要开机连续点击`F10`进入`bios`，选择安全，查看`TPM选项开启`，如果没有TPM需要联系维修站查看或者联系线上人工查询配置

安装过程:

1. 首先将制作好的镜像U盘插入到电脑USB接口中，开机连续点击键盘`F9`键，在打开的菜单中，选择`USB硬盘设备`，点击回车键。

2. 在打开的windows安装页面中，选择`下一页`。

3. 点击`现在安装`。

4. 勾选`我接受Microsoft 软件许可条款`后，点击`下一页`

5. 选择`自定义：仅安装windows（高级）`

6. 在系统分区页面中，我们需要将之前的系统分区全部删掉，分别选择每一个分区，点击页面底部的`删除`即可。（如果硬盘中有重要数据，请提前备份）

    注：如果此界面是空的，没有显示硬盘，且机器的cpu是11代以上，查看以下具体解决方法

    <details>
      <summary>如何加载硬盘驱动</summary>
      - 【解决方案】
      当前系统无法进入的时候，请借助另一台电脑下载驱动，并将驱动文件拷贝到U盘里

      1. 下载驱动：
      链接：https://ftp.hp.com/pub/softpaq/sp134501-135000/sp134521.exe

      2. 在已下载的`sp134521`驱动图标上，单击`鼠标右键`，选择`压缩软件`（具体名字与电脑安装的压缩软件有关）--`解压缩`

      3. 选择`U盘`，然后点击`解压缩`。
      注：如果电脑没有压缩软件，可直接双击`sp134521`驱动图标，点击`下一步`--`接受条款`，在保存路径上，点击`change…`

      4. 此时驱动文件已经全部拷贝到U盘，然后重新回到之前安装系统无法识别到硬盘的界面。

      5. 在分区页面直接点击`加载驱动程序`，点击`浏览`，选择U盘，选中`dchu_VMD`文件夹，选择`Intel RST vmd controller`,点击【下一步】，加载完成后即可识别到硬盘继续安装系统操作。
    </details>

7. 提示删除分区会导致所有数据丢失，点击`确定`

8. 删除完成后，我们选择`驱动器0未分配的空间`然后点击`下一页`

9. 出现windows正在安装页面，我们稍等即可，安装完成后，机器会自动重启。

10. 出现即将重启提醒页面，我们可以等待读秒结束，也可以点击`立即重启`选项即刻重启。

11. 重启后，提示`准备就绪`，此时就可以拔掉U盘稍作等待了。
