# 基本用法
```scss
.container{
  display: grid;
  // 表示自动填充列数，每一列最小为250px，最大为 1 / 列数n
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  // 列间距
  grid-column-gap: 1rem;
  // 行间距
  grid-row-gap: 2rem;
}
```
MDN文档： [https://developer.mozilla.org/en-US/docs/Web/CSS/grid](https://developer.mozilla.org/en-US/docs/Web/CSS/grid)
